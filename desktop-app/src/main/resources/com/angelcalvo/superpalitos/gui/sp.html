2. CHANGE-LOG

Esta es la milestone 3 de SuperPalitos 3, que es la penúltima antes de la versió final, en el que se incluirán modos de juego.

Cambios en la version 3.0 M3

  • Nueva arquitectura PalitosNet que permite jugar varias partidas en red
  • Pequeñas mejoras gráficas
  • Creado el sonido del boli

Cambios en la version 3.0 M2

  • Pestañas!
  • Animaciones
  • Sonido
  • Ahora funciona con JWS (Java Web Start).
  • Cambios graficos menores y correción de pequeños bugs.

Cambios en la version 2.2

  • El chat ahora muestra los mensajes con el color del jugador.
  • Arreglado un bug en la IA que hacia que la partida terminase cuando esta iba a perder.
  • Ahora YA funciona con J2SE 1.4.
  • Algunos cambios gráficos menores.

Cambios de la version 2.1 respecto de la versión 2.0

  • Ahora Super Palitos es LIBRE!
  • Se ha corregido el bug que al mostrar el gandor de una partida mostraba el perdedor.
  • Se ha corregido el bug que hacía que el juego se jodiese al abortar una partida en curso.
  • Se ha mejorado la clase IA.
  • Se ha reescrito la clase PalitosNet.
  • Se ha incluido el submenu PalitosNet al menu Preferencias.
  • Se ha incluido el submenu Licencia al menu Acerca de...
  • Ahora a pulsar enter se manda un mensaje en el chat.
  • Se han corregido otros bugs menores.

3. REQUISITOS

Para que el juego funcione se necesita tener instalado la JRE 1.4 o superior.

5. COMO JUGAR

Para arrancar la aplicación hacer doble click en el fichero superPalitos-3.0.m3.jar o desde una consola ejecutar el comando: 

 java -jar superPalitos-3.0.m3.jar

El juego consiste en ir tachando palitos por turnos. El objetivo del jugador es dejar solo un palito. Se pueden tachar todos los palitos que se quieran de una fila siempre que no esten ya tachados.
